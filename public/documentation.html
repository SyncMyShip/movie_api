myFlix

Overview

The purpose of this Achievement is to build out the server-side component
of a web application. This application will be dedicated to providing users
access to information about:

  • Movies
  • Directors
  • Genres
  • And more...

Functionality

Once the application has been deployed, users will have the freedom to:

  • Sign up for myFlix
  • Update personal account details
  • Create a list of their favorite movies
